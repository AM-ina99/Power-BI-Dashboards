## 📌 Overview
This project presents a comprehensive **Hospital Management Dashboard** built using **Power BI**.  
It provides insights into hospital operations, patient distribution, financial performance, and time-based trends to support data-driven decision-making.

The dashboard is divided into multiple analytical views, enabling stakeholders to monitor KPIs and improve operational efficiency.

---

## 🎯 Objectives
- Monitor hospital performance through key KPIs
- Analyze patient demographics and medical conditions
- Track hospital operations over time
- Evaluate financial performance and revenue streams
- Support strategic decision-making in healthcare management

---

## 📊 Dashboard Pages

### 1️⃣ Home Page
- Navigation hub for all dashboard sections
- Quick access to:
  - Hospital Overview
  - Patient Insights
  - Time Analysis
  - Financial Analysis

---

### 2️⃣ Hospital Overview

### 3️⃣ Patient Insights
Focuses on patient demographics and medical data:

**Key Analysis:**
- Patients by gender (Female, Male, Other)
- Patients by age groups
- Patients by disease (e.g., Flu, Diabetes, Cancer)
- Disease vs Gender analysis
- Surgeries by gender
- Patients distribution by state

**Filters Available:**
- Patient Name
- Gender
- Room Number
- Age Group

---
